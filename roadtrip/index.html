The Ultimate Climbing Roadtrips

Have you ever wondered how to link together the best climbing destinations to create the Ultimate Climbing Roatrip? One way to answer that question is to use the MountainProject.com database to identify the best climbing destinations in each region and the GoogleMaps API to find the optimal driving route that links them together. That's what I did here. Picking the best climbing destinations is always a matter of debate, but for this analysis I used a combination of popularity and number of routes. Finding the optimal driving route is also tricky (see Travelling Salesman Problem). Thankfully, the GoogleMaps API will do it for you when there are less than 10 destinations. So if you want to know the optimal driving route that links all 50 destinations, see your local computer scientist.
